# Getting Started

Welcome to ollog — a self-hosted, multi-operator ham radio logbook. This section walks you through everything you need to go from zero to logging QSOs.

ollog is designed to be deployed with Docker Compose in minutes. Once running, each licensed operator can log QSOs independently under their own callsign, with no risk of mixing up logs between operators.

## In This Section

- [Quickstart](quickstart.md) — Deploy the stack and verify the API is up
- [First QSO](first-qso.md) — Log your first contact via the web UI and API

## Prerequisites

Before you begin, you will need:

- Docker and Docker Compose v2 installed on your host
- Port 8000 available on that host
- An admin callsign and credentials to use as the bootstrap admin account
