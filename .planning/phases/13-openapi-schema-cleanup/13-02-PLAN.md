  <name>Task 1: Define ADIFImportReport model and annotate ADIF endpoints</name>
  <files>app/adif/router.py</files>
  <action>
Add Pydantic models and annotate the two ADIF endpoint decorators in `app/adif/router.py`.

1. **Add imports:** Add `BaseModel`, `Optional` from pydantic (BaseModel may need import; check existing imports).

2. **Define sub-models and ADIFImportReport** (add after imports, before `_REQUIRED_FIELDS`):

   ```python
   class ADIFRecordAccepted(BaseModel):
       record_index: int
       call: str
       id: str

   class ADIFRecordDuplicate(BaseModel):
       record_index: int
       call: str
       existing_id: str

   class ADIFRecordError(BaseModel):
       record_index: int
       call: Optional[str] = None
       error: str

   class ADIFImportReport(BaseModel):
       total_records: int
       accepted: list[ADIFRecordAccepted]
       duplicates: list[ADIFRecordDuplicate]
       errors: list[ADIFRecordError]
   ```

   IMPORTANT: `call` in ADIFRecordError is `Optional[str] = None` because parse errors from `parse_adi()` have `record_index` and `error` keys but NO `call` key (see parser.py lines 78-81). Per-record errors from `process_import()` DO include `call`. The optional field handles both shapes.

3. **Annotate import endpoint** — add `response_model=ADIFImportReport`:
   ```python
   @router.post("/import", response_model=ADIFImportReport)
   async def import_adif(
       file: UploadFile,
       operator: str = Depends(get_current_operator_callsign),
   ):
   ```

4. **Annotate export endpoint** — add `response_class` and `responses` for text/plain:
   ```python
   @router.get(
       "/export",
       response_class=StreamingResponse,
       responses={
           200: {
               "content": {"text/plain": {}},
               "description": "ADIF (.adi) file download containing the operator's full logbook.",
           }
       },
   )
   async def export_adif(
       operator: str = Depends(get_current_operator_callsign),
   ):
   ```
   StreamingResponse is already imported. Do NOT add `response_model` to the export endpoint — StreamingResponse cannot be validated by Pydantic.

Do NOT change `process_import()`, `_qso_to_adif_dict()`, or any function body — only models and decorators change.
