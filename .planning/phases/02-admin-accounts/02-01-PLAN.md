<task type="auto">
  <name>Task 1: Admin API router with create, toggle-enabled, and reset-password endpoints</name>
  <files>app/admin/__init__.py, app/admin/router.py, app/main.py</files>
  <action>
Create app/admin/__init__.py (empty or minimal).

Create app/admin/router.py with APIRouter(prefix="/admin/users", tags=["admin"]):

Request models (Pydantic BaseModel):
- CreateUserRequest: username (str), callsign (str), password (str)
- ResetPasswordRequest: password (str)
- SetEnabledRequest: enabled (bool)

Endpoints — ALL use `Depends(require_admin)` from app.auth.dependencies:

1. POST /admin/users/ (status_code=201):
   - Accept CreateUserRequest body
   - Check if username already exists — if so, raise 409 Conflict with detail "Username already exists"
   - Create User with role="operator", enabled=True, callsign uppercased, hashed password via hash_password()
   - Return {"username": user.username, "callsign": user.callsign, "enabled": user.enabled, "role": user.role}

2. PATCH /admin/users/{username}/enabled:
   - Accept SetEnabledRequest body (with `enabled: bool`)
   - Look up user by username — 404 if not found
   - LAST-ADMIN GUARD: If disabling (enabled=False) AND target user has role="admin", count how many admins have enabled=True. If count <= 1, raise 409 Conflict with detail "Cannot disable the last enabled admin"
   - Use `await user.set({User.enabled: body.enabled})` (field expression form, NOT string keys)
   - Return {"username": username, "enabled": body.enabled}

3. POST /admin/users/{username}/reset-password:
   - Accept ResetPasswordRequest body
   - Look up user by username — 404 if not found
   - Use `await user.set({User.hashed_password: hash_password(body.password)})` (field expression form)
   - Return {"username": username, "password_reset": True}

4. GET /admin/users/ (list all users):
   - Return list of all users with fields: username, callsign, role, enabled
   - Do NOT return hashed_password
   - Use `await User.find_all().to_list()`

Update app/main.py:
- Import admin router: `from app.admin.router import router as admin_router`
- Add `app.include_router(admin_router)` after the auth router include

CRITICAL: Use Beanie field expression form `{User.enabled: value}` in .set() calls, NOT string keys `{"enabled": value}`.
CRITICAL: The last-admin guard must count enabled admins, not just check the current user. An admin could disable a different admin.
  </action>
