Wire the full UDP QSO processing pipeline: parse ADIF datagrams, validate required fields, build QSO documents with operator attribution and profile auto-stamping, detect duplicates, and insert into MongoDB.

Output: Working `_handle_datagram()` coroutine in `app/udp/server.py`, updated lifespan in `app/main.py` with cached User lookup, and unit tests in `tests/test_udp_pipeline.py`.
</objective>

```python
udp_transport = None
if settings.udp_enabled:
    from app.auth.models import User as UserModel
    from app.udp.server import start_udp_listener

    udp_user: UserModel | None = None
    udp_op: str | None = settings.udp_operator
    if udp_op:
        udp_op = udp_op.upper()  # normalise callsign casing
        udp_user = await UserModel.find_one({"callsign": udp_op})
        if udp_user is None:
            logger.warning(
                "UDP_OPERATOR callsign %r not found in DB ŌĆö profile stamping disabled",
                udp_op,
            )

    udp_transport, _ = await start_udp_listener(
        settings.udp_bind_host,
        settings.udp_port,
        operator=udp_op,
        user=udp_user,
    )
```
