## No UI Changes in This Phase

Phase 51 adds two plain-text GET endpoints (`/llms.txt` and `/llms-full.txt`) backed by
static files in `static/`. There are no HTML templates, no CSS changes, no form elements,
no new components, and no visual changes of any kind.

This UI-SPEC.md exists solely to satisfy the UI gate requirement. The checker should
record all six dimensions as not-applicable and approve immediately.

**Scope of changes:**

| Artifact | Changed? | Notes |
|----------|----------|-------|
| HTML templates | No | No new or modified templates |
| CSS / Tailwind | No | No new classes; no output.css rebuild needed |
| Form elements | No | No forms added |
| Interactive components | No | No HTMX, no SSE, no JS |
| Static files (visual) | No | `static/llms.txt` and `static/llms-full.txt` are plain text, not CSS/JS/images |
| FastAPI routes | Yes (backend only) | Two `FileResponse` routes, `include_in_schema=False` |
