```python
app.mount("/static", StaticFiles(directory="static"), name="static")


# llms.txt endpoints — static content served from disk via FileResponse.
# Routes are excluded from the OpenAPI schema (LLMS-03).
# Editing static/llms.txt or static/llms-full.txt and restarting the app
# serves updated content with no Python code change (LLMS-04).
@app.get("/llms.txt", include_in_schema=False)
async def llms_index():
    """LLM tooling index — project title, description, section links."""
    return FileResponse(
        path="static/llms.txt",
        media_type="text/plain; charset=utf-8",
    )


@app.get("/llms-full.txt", include_in_schema=False)
async def llms_full():
    """Full LLM reference — API docs, ADIF field guide, getting started."""
    return FileResponse(
        path="static/llms-full.txt",
        media_type="text/plain; charset=utf-8",
    )


@app.exception_handler(HTTPException)
```

**Critical rules:**
- Do NOT add auth dependencies — these are public endpoints (matching `/health`).
- Do NOT add `filename=` parameter to FileResponse — we want the browser to render the content, not trigger a download (unlike `backup_download` in `app/admin/ui_router.py`).
- `media_type="text/plain; charset=utf-8"` MUST be passed explicitly — RESEARCH.md Pitfall 2 shows that auto-detection from the `.txt` extension omits `; charset=utf-8` and fails LLMS-03.
- `include_in_schema=False` goes on the `@app.get(...)` decorator (per Context7 FastAPI docs), NOT elsewhere.
- The path argument to FileResponse is the RELATIVE path `"static/llms.txt"` (not an absolute path, not a `pathlib.Path`) — matches the `StaticFiles(directory="static")` convention at line 179 and works because uvicorn runs from the project root in both Docker and `uv run pytest`.
- Two blank lines between top-level functions (PEP 8). Match the existing style in `app/main.py`.
