Create app/qso/service.py with:

1. `parse_adif_datetime(qso_date: str, time_on: str) -> datetime`:
   - Parse QSO_DATE (YYYYMMDD) and TIME_ON (HHMM or HHMMSS) into a UTC-aware datetime.
   - `date_part = datetime.strptime(qso_date, "%Y%m%d").date()`
   - `time_part = datetime.strptime(time_on, "%H%M").time()` if len==4, else `"%H%M%S"`
   - Return `datetime.combine(date_part, time_part, tzinfo=timezone.utc)`

2. `build_qso_dict(body_dict: dict, operator: str) -> dict`:
   - Start with a copy of body_dict (all ADIF fields from request).
   - Uppercase BAND and MODE if present.
   - Parse QSO_DATE + TIME_ON into qso_date_utc and add to dict.
   - Add `operator_callsign` key set to operator (Beanie will serialize to `_operator`).
   - Add `is_deleted` key set to False (Beanie will serialize to `_deleted`).
   - Remove QSO_DATE and TIME_ON from the dict if you store them separately, OR keep them for ADIF round-trip fidelity. KEEP THEM — Phase 4 needs them for export.
   - Return the dict ready for QSO(**dict) construction.

3. `get_qso_page(operator, page, page_size, callsign_filter, band_filter, mode_filter, date_from, date_to, sort_by) -> tuple[list[QSO], int]`:
   - Build query dict: `{"_operator": operator, "_deleted": False}`
   - Add optional filters: CALL regex (case-insensitive), BAND exact, MODE exact, qso_date_utc $gte/$lte.
   - Count total via `await QSO.find(query).count()`
   - Fetch page via `await QSO.find(query).sort(sort_by).skip((page-1)*page_size).limit(page_size).to_list()`
   - Return (items, total).
   - Default sort_by: "-qso_date_utc" (newest first).
   - Do NOT use find_active() — it returns a list, not a query builder.

Create app/qso/router.py with APIRouter(prefix="/api/qsos", tags=["qsos"]):

Request model:
- QSOCreateRequest: BaseModel with `model_config = ConfigDict(extra="allow")`. Declared fields: CALL (str), QSO_DATE (str), TIME_ON (str), BAND (str), MODE (str), FREQ (Optional[str]=None), RST_SENT (Optional[str]=None), RST_RCVD (Optional[str]=None). Extra ADIF fields captured via model_extra.

Endpoints — ALL use `Depends(get_current_operator_callsign)`:

1. POST /api/qsos/ (status_code=201):
   - Accept QSOCreateRequest body and optional `force: bool = Query(False)` param (for duplicate override — wired in 03-02, ignored for now).
   - Call build_qso_dict(body.model_dump(exclude_unset=False) | (body.model_extra or {}), operator).
   - NOTE on model_dump: The declared fields come from model_dump(). The extra ADIF fields are in body.model_extra. Merge them: `{**body.model_dump(exclude_unset=False), **(body.model_extra or {})}`. Then pass to build_qso_dict.
   - Construct QSO(**qso_dict) and await qso.insert().
   - Return the QSO as dict via `qso.model_dump(by_alias=True)` with `str(qso.id)` as `id` field.

2. GET /api/qsos/ (paginated list):
   - Query params: page (int=1), page_size (int=50), call (Optional[str]), band (Optional[str]), mode (Optional[str]), date_from (Optional[str] YYYYMMDD), date_to (Optional[str] YYYYMMDD), sort (str="-qso_date_utc").
   - Parse date_from/date_to strings to UTC datetimes if provided.
   - Call get_qso_page(operator, page, page_size, ...).
   - Return {"items": [...], "total": total, "page": page, "page_size": page_size}.
   - Each item: qso.model_dump(by_alias=True) with id as string.

3. GET /api/qsos/{qso_id}:
   - Fetch by ID. Return 404 if not found, not owned by operator, or soft-deleted.
   - Return QSO as dict.

4. PATCH /api/qsos/{qso_id}:
   - Accept body: Dict[str, Any].
   - Fetch QSO by ID. 404 if not found, not owned, or deleted.
   - Strip protected fields: _operator, operator_callsign, _deleted, is_deleted, _id.
   - If QSO_DATE or TIME_ON in body, re-parse qso_date_utc from the combined values (use existing values for whichever is not provided).
   - If BAND in body, uppercase it. If MODE in body, uppercase it.
   - Use `await qso.update({"$set": body})` — raw $set works for both declared fields and model_extra ADIF keys.
   - Refetch and return updated QSO.

5. DELETE /api/qsos/{qso_id} (status_code=204):
   - Fetch QSO by ID. 404 if not found, not owned, or already deleted.
   - Use `await qso.update({"$set": {"_deleted": True}})` — use MongoDB alias _deleted, NOT Python attribute is_deleted.
   - Return 204 No Content.

CRITICAL:
- Use MongoDB alias names in ALL raw $set dicts: "_deleted" not "is_deleted", "_operator" not "operator_callsign".
- Never accept operator callsign from request body — always from JWT dependency.
- Uppercase BAND and MODE on ingest for compound index consistency.
- Use `extra="allow"` on QSOCreateRequest for ADIF passthrough to Phase 4.
