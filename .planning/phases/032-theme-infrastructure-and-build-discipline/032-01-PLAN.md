In `templates/base.html`:

1. Add color-scheme meta tag as the FIRST meta tag after `<meta name="viewport">` (line 5):
   ```html
   <meta name="color-scheme" content="light dark">
   ```

2. Move the IIFE script block to BEFORE the `<link rel="stylesheet">` tag (currently line 10). The IIFE must appear before `<link rel="stylesheet" href="/static/css/output.css">` for correctness.

3. Replace the single-line comment with a multi-line LOAD-BEARING annotation:
   ```html
   <!--
     LOAD-BEARING: This script block MUST remain:
     - Inline (not src="..."), so the browser executes it synchronously
     - In <head>, before the <link rel="stylesheet"> tag
     - Non-deferred (no defer or async attribute)
     Moving, deferring, or externalising this script will cause a FOUC
     (flash of unstyled/light content) for dark-mode users on every page load.
   -->
   ```

4. Replace the IIFE body with the rAF-rAF transition suppression pattern:
   ```js
   (function () {
     var noTransition = document.createElement('style');
     noTransition.textContent = '*, *::before, *::after { transition: none !important; }';
     document.head.appendChild(noTransition);

     var theme = localStorage.getItem('theme');
     if (theme === 'dark' || (!theme && window.matchMedia('(prefers-color-scheme: dark)').matches)) {
       document.documentElement.classList.add('dark');
     }

     requestAnimationFrame(function () {
       requestAnimationFrame(function () {
         document.head.removeChild(noTransition);
       });
     });
   })();
   ```
   The logic is identical to the existing IIFE (localStorage check, system preference fallback, add 'dark' class). The additions are: inject a `transition: none !important` style before applying dark, then remove it after two rAFs so user-initiated toggles still animate.

In `static/css/input.css`:

5. In the existing `@layer base` block, update the `html` rule to add `color-scheme: light;` and add a new `html.dark` rule with `color-scheme: dark;`. The existing `font-family` on `html` stays. Result:
   ```css
   @layer base {
     html {
       font-family: 'Inter', ui-sans-serif, system-ui, -apple-system, sans-serif;
       color-scheme: light;
     }
     html.dark {
       color-scheme: dark;
     }
     /* Default style for bare <button> elements ... (unchanged) */
   }
   ```

Do NOT add `transition-*` to `html`, `body`, or `*` in `@layer base`. Do NOT remove any existing CSS rules.
