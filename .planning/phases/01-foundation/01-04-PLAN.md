Create app/auth/models.py with User Beanie Document:
- Fields: username (str, unique), hashed_password (str), callsign (str), role (str, default "operator" — values: "operator" or "admin"), enabled (bool, default True)
- class Settings: name = "users", indexes with unique index on username
- No extra="allow" needed — User schema is fixed

Create app/auth/service.py:
- Import jwt from pyjwt (import jwt, NOT from jose import jwt)
- Import PasswordHash from pwdlib (NOT passlib)
- password_hash = PasswordHash.recommended()  # Argon2 per research recommendation
- hash_password(plain: str) -> str: return password_hash.hash(plain)
- verify_password(plain: str, hashed: str) -> bool: return password_hash.verify(plain, hashed)
- create_access_token(data: dict, expires_delta: timedelta | None = None) -> str:
  - data should include: sub (username), callsign, role
  - Set exp claim using settings.jwt_expire_minutes if expires_delta not provided
  - Encode with settings.secret_key and settings.jwt_algorithm
  - Return encoded JWT string
- decode_access_token(token: str) -> dict:
  - Decode with settings.secret_key and settings.jwt_algorithm
  - Raise InvalidTokenError on failure
  - Return payload dict

Create app/auth/dependencies.py:
- oauth2_scheme = OAuth2PasswordBearer(tokenUrl="/auth/token")
- async def get_current_user(token: str = Depends(oauth2_scheme)) -> User:
  - Decode token via decode_access_token
  - Extract username from "sub" claim
  - Look up User by username
  - If not found or not enabled, raise 401
  - Return User object
- async def get_current_operator_callsign(user: User = Depends(get_current_user)) -> str:
  - Return user.callsign — THIS is the callsign injection point. All QSO operations will depend on this, never on request body.
- async def require_admin(user: User = Depends(get_current_user)) -> User:
  - If user.role != "admin", raise 403 Forbidden
  - Return user

Update app/database.py:
- Import User from app.auth.models
- Add User to document_models list alongside QSO

Update app/auth/__init__.py to export key items.
