Create `tests/test_operator_isolation.py` with a route introspection test that programmatically verifies every QSO-related endpoint has a callsign injection dependency.

**Test: `test_all_qso_routes_inject_callsign_from_jwt`**

1. Import the FastAPI `app` from `app.main`.
2. Import `APIRoute` from `fastapi.routing`.
3. Define the set of callsign dependency names:
   ```python
   CALLSIGN_DEPS = {
       "get_current_operator_callsign",
       "get_current_operator_callsign_cookie",
   }
   ```
4. Define QSO-related path prefixes that MUST have callsign injection:
   ```python
   QSO_PATH_PREFIXES = ("/api/qsos", "/log/qsos", "/log/view", "/log/import", "/log/export", "/api/adif")
   ```
5. Exclude paths that are login/logout (don't need callsign): `/log/login`, `/log/logout`.
6. For each `APIRoute` in `app.routes` whose path starts with any QSO prefix and is not in the exclude list:
   - Use `inspect.signature(route.endpoint)` to get parameters.
   - For each parameter, check if `param.default` has a `dependency` attribute with `__name__` in `CALLSIGN_DEPS`. Also check if the parameter name itself suggests callsign injection (e.g., parameter annotated with Depends).
   - A simpler approach: collect the names of all callables in the route's dependency chain. Use the research pattern:
     ```python
     import inspect
     sig = inspect.signature(route.endpoint)
     dep_names = set()
     for param in sig.parameters.values():
         default = param.default
         if hasattr(default, 'dependency'):
             dep_names.add(default.dependency.__name__)
         elif callable(default) and hasattr(default, '__name__'):
             dep_names.add(default.__name__)
     ```
   - Assert `dep_names & CALLSIGN_DEPS` is non-empty for this route. If it fails, the error message must include the route path and methods.
7. Also verify the route count is at least 8 (to guard against the test silently passing because no routes matched).

This test catches any future endpoint that forgets the callsign dependency.
  </action>
