<task type="auto">
  <name>Task 1: Create ADIF import API endpoint</name>
  <files>app/adif/router.py, app/adif/__init__.py, app/main.py</files>
  <action>
Create `app/adif/router.py` with an APIRouter at prefix `/api/adif`, tags `["adif"]`.

Add `POST /import` endpoint:
- Accepts `file: UploadFile` (use `from fastapi import UploadFile`)
- Depends on `get_current_operator_callsign` for operator injection from JWT
- Read file bytes: `raw = await file.read()`
- Check `len(raw) > 10 * 1024 * 1024` — raise HTTPException 413 "File exceeds 10 MB limit"
- Decode: `text = raw.decode("utf-8", errors="replace")` — handles Latin-1 ADIF files gracefully
- Parse: `records, parse_errors = parse_adi(text)` from `app.adif.parser`
- Initialize: `accepted = []`, `duplicates = []`, `errors = list(parse_errors)`
- Loop over `enumerate(records)`:
  - Check required fields: `{"CALL", "QSO_DATE", "TIME_ON", "BAND", "MODE"} <= set(record)`. If missing, append to errors with `record_index`, `error: "Missing required field(s)"`, `call: record.get("CALL", "?")`. Continue.
  - Try `qso_dict = build_qso_dict(record, operator)`. On ValueError/KeyError, append to errors with `record_index`, `error: str(exc)`, `call: record.get("CALL", "?")`. Continue.
  - (NO duplicate detection in this plan — 04-02 adds it here)
  - Create and insert: `qso = QSO(**qso_dict); await qso.insert()`
  - Append to accepted: `{"record_index": idx, "call": qso_dict["CALL"], "id": str(qso.id)}`
- Return dict: `{"total_records": len(records), "accepted": accepted, "duplicates": duplicates, "errors": errors}`

Register the router in `app/main.py`: import and `app.include_router(adif_router)` — place it after the QSO UI router, before the static files mount. Follow the existing import pattern with noqa comments.

Ensure `app/adif/__init__.py` exists (it already does, but verify it allows the import).
  </action>
  <verify>
Run: `python -c "from app.adif.router import router; print('OK')"` — imports without error.
Run existing tests to confirm no regressions: `python -m pytest tests/ -x -q`
  </verify>
  <done>POST /api/adif/import endpoint exists, accepts UploadFile, parses ADIF, validates required fields, inserts accepted records, returns complete JSON report with accepted/duplicates/errors counts. Router registered in main.py.</done>
