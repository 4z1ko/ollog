## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-53-01 | Tampering | QSO_DATE / TIME_ON inputs | accept | The fields use `readonly` (not `disabled`) so values reach the server.
